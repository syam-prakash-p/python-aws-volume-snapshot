# python-aws-volume-snapshot
#### requirements
```sh
$ pip3 install boto3
```
``` sh
vim ~/.aws/config
[default]
region= your_required_region
```
``` sh
vim ~/.aws/credentials
[default]
aws_access_key_id = your_access_key
aws_secret_access_key = your_secret_key
```
##### running program

```sh

python3 snapshot.py
```
